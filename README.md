## Deskripsi
Aplikasi web CRUD untuk manajemen kategori buku perpustakaan menggunakan PHP dan MySQL.

## Cara Instalasi
1. Clone repository ini ke folder `htdocs` XAMPP
2. Import file `database/database_backup.sql` ke phpMyAdmin
3. Sesuaikan nama database di `config/database.php` jika perlu
4. Akses via `http://localhost/uts_60324038/`
